# Flask Student Management System

A Student Management System developed using Flask and MySQL. This project allows users to add, view, search, update, and delete student records through a web-based interface.

## Features

* User Login Authentication
* Add New Students
* View Student Records
* Search Students by Roll Number
* Update Student Information
* Delete Student Records
* Dashboard with Total Student Count

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS

## Running the Project

1. Install the required libraries.
2. Configure the MySQL database.
3. Run:

python app.py

4. Open:

http://127.0.0.1:5000

## Project Purpose

This project was built to practice Flask development, database integration, CRUD operations, and web application development using Python.
